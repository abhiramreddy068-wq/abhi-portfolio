# Abhiram Reddy - Portfolio

A professional portfolio website showcasing my skills in Python and AWS, built with HTML and CSS, deployed to AWS S3 with GitHub Actions CI/CD.

**Live Portfolio:** [View Here](http://abhiram-portfolio.s3-website-us-east-1.amazonaws.com)

## Features

- Responsive design with modern UI
- Showcase of Python and AWS projects
- Education and skills section
- Automated deployment to S3 via GitHub Actions

## Tech Stack

- HTML5
- CSS3
- AWS S3 (Static Website Hosting)
- GitHub Actions (CI/CD)

## Projects Included

1. **Weather Forecast Application** - Flask + OpenWeatherMap API
2. **Task Management System** - Django + SQLite
3. **AWS S3 File Manager** - Python + Boto3
4. **Data Analysis Dashboard** - Streamlit + Pandas

## Deployment

This portfolio is automatically deployed to AWS S3 on every push to the `main` branch using GitHub Actions.

See [DEPLOYMENT.md](DEPLOYMENT.md) for setup instructions.

## Local Development

1. Clone the repository
2. Open `portfolio.html` in your browser
3. Make changes and push to trigger automatic deployment
